# 5-Tage-Agenda: HTML / CSS / JS / TS / React

**Tag 1: Grundlagen HTML & CSS**  
Struktur, Tags, Semantik, CSS-Styling, Flexbox und Grid.

**Tag 2: JavaScript (JS)**  
Variablen, Funktionen, DOM-Manipulation und Events.

**Tag 3: TypeScript (TS) & React Startup**  
Typen, Interfaces, Typ-Sicherheit und Initialisierung eines Projekts mit Vite.

**Tag 4: React-Grundlagen**  
Komponenten, Props, State, Hooks und JSX/TSX.

**Tag 5: Praxisprojekt**  
Entwicklung einer App (z. B. To-Do-Liste) mit React & TypeScript.

---

### Tag 1: Grundlagen von HTML & CSS
- **HTML-Struktur**: Erstellen Sie eine grundlegende Webseite mit `<html>`, `<head>`, `<body>` und Elementen wie `<h1>`, `<p>`, `<ul>`, `<img>`. Beispiel: Ein einfaches Portfolio mit Navigation und Footer.
- **CSS-Styling**: Lernen Sie Selektoren (`.class`, `#id`, `element`) und Eigenschaften wie `color`, `font-size`, `margin`, `padding`. Erstellen Sie ein responsives Design mit Flexbox oder Grid.
- **Praxisbeispiel**: Erstellen Sie eine Landingpage mit einem Call-to-Action-Button. Nutzen Sie CSS-Animationen (z. B. `transition`) für interaktive Effekte.
- **Übungen**: Erstellen Sie eine Tabelle mit CSS-Layout, experimentieren Sie mit Farbpaletten und Schriftarten.

**Tag 2: JavaScript-Grundlagen & DOM-Manipulation**
- **JavaScript-Syntax & Datenstrukturen**: Variablen (`let`, `const`), dynamische Typisierung (`===`), Objekt-Literale und Arrays inkl. Methoden (`push`, `map`, `filter`).
- **DOM-Interaktion**: Nutzen Sie JavaScript, um HTML-Elemente zu selektieren und zu manipulieren (z. B. `querySelector`, `textContent`, `classList`).
- **Events & Interaktivität**: Reaktion auf Benutzeraktionen (z. B. `click`, `keyup`, `submit`). Erstellung eines dynamischen Menüs.
- **Asynchrones JavaScript**: Einführung in Callbacks, Promises und `async/await`. Umgang mit `setTimeout` und API-Aufrufen (`fetch`).
- **Praxisbeispiel**: Erstellen Sie eine funktionale Taschenrechner-App. Nutzen Sie Event-Handling und die DOM-API für Eingaben und Berechnungen.
- **Übungen**: Arbeiten mit Arrays und Objekten, DOM-Manipulation in Echtzeit (Uhrzeit, To-Do-Liste) und asynchrone Datenabfragen.

### Tag 3: TypeScript & Projekt-Setup
- **TypeScript-Grundlagen**: Erweitern Sie JavaScript mit Typen (`string`, `number`), Interfaces und Enums. Beispiel: Ein Interface für ein Benutzer-Objekt.
- **Vite & React Startup**: Erstellen Sie ein neues Projekt mit Vite:  
  `npm create vite@latest mein-projekt -- --template react-ts`
- **Praxisbeispiel**: Entwickeln Sie eine To-Do-Liste mit TypeScript-Interfaces für die Aufgabenstruktur.
- **Übungen**: Schreiben Sie eine Funktion, die ein Objekt mit `interface` validiert. Experimentieren Sie mit Generics.

### Tag 4: React-Grundlagen & Komponenten
- **Komponenten**: Erstellen Sie funktionale Komponenten und nutzen Sie Props. Beispiel: Eine `Header`-Komponente mit Titel und Navigation.
- **JSX & TSX**: Verstehen Sie die Syntax von React in Verbindung mit TypeScript (`.tsx`).
- **Hooks**: Nutzen Sie `useState` und `useEffect` für interaktive Elemente. Beispiel: Ein Zähler, der bei Mount-Event lädt.
- **Praxisbeispiel**: Erstellen Sie eine Wetter-App, die Daten von einer API abruft und in einer `WeatherCard`-Komponente anzeigt.
- **Übungen**: Schreiben Sie eine `TodoList`-Komponente, die Aufgaben aus einem State-Array rendert.

### Tag 5: Projektentwicklung & Integration
- **Projekt-Integration**: Kombinieren Sie HTML, CSS, JS, TS und React. Beispiel: Eine Portfolio-Website mit React und TypeScript.
- **Backend-Anbindung (optional)**: Nutzen Sie APIs (z. B. Fetch-API, Firebase) für Datenabfragen.
- **Optimierung**: Implementieren Sie Responsivität und nutzen Sie ggf. `useContext` für die Zustandsverwaltung.
- **Übungen**: Erstellen Sie ein Portfolio mit Dark-Mode-Toggle. Nutzen Sie TypeScript für den Zustand.
- **Ressourcen**: Dokumentationen (MDN, React Docs), VS Code, Chrome DevTools.
