# 🎨 Übung: CSS Grundlagen

## Aufgabe: Toben Sie sich auf Ihrer Landingpage aus 🦄

Gestalten Sie Ihre Landingpage für Ihr Lernprodukt mit CSS. Hier können Sie aus dem Vollen schöpfen. Gerade bei zwielichtigen oder unrealistischen Produkten wird oft ein Design gewählt, das die Aufmerksamkeit der Website-Besucher massiv auf sich zieht.

### Anforderungen

- Flippige Farben.
- Riesige und sehr kleine Schriftarten.
- Verschiedene Schriftarten (Google Fonts).
- Hintergrundgestaltung (Styles für den `<body>`).
- Verwenden Sie ein externes Stylesheet (`style.css`).

**Tipp:** Wer die am schrecklichsten aussehende Website erstellt, gewinnt! Fokus auf Selektoren und Text-Eigenschaften.

### Für Neugierige

- Experimentieren Sie mit: `background-color`, `color`, `font-family`, `font-size`, `font-weight`, `text-shadow`, `text-decoration`.
- Fortgeschrittene Eigenschaften: `text-transform`, `text-align`, `line-height`, `letter-spacing`.
- Versuchen Sie, bestimmte Teile Ihrer Webseite zu animieren.

## Aufgabe: Klassen- und ID-Selektoren

1. **Teil 1:** Verwenden Sie verschiedene Klassen, um Elemente zu gestalten, und überschreiben Sie einen Style, indem Sie ein bestimmtes Element über seine ID ansprechen.
2. **Teil 2:** Überfliegen Sie diese Liste von Selektor-Kombinationen ([W3Schools CSS Combinators](https://www.w3schools.com/css/css_combinators.asp)), wählen Sie mindestens zwei aus und wenden Sie diese an.
