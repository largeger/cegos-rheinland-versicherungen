# 👨‍💻 Übung: Produkt-Landingpage

Wenn Sie sich an die Einstiegsfrage von vorhin erinnern, ist klar, dass man außerordentlich viel Zeit benötigen würde, um wirklich alles über ein Fachgebiet zu wissen. Das ist jedoch oft nicht das, was man im Internet liest, wenn es um bestimmte Lernprodukte geht, die verkauft werden.

"Lernen Sie Java in nur 2 Tagen und sichern Sie sich einen Job bei Google mit unserem fantastischen Lernprodukt."

"Werden Sie in nur einem Jahr Millionär mit diesem eBook."

## Aufgabe: Erstellung der Landingpage

Erstellen Sie eine Landingpage für ein "zwielichtiges" Lernprodukt (nur HTML). Als Besucher Ihrer Homepage möchte ich schnell ein Gefühl dafür bekommen, worum es bei dem Produkt geht.

### Anforderungen

- Ein Titel und eine Überschrift.
- Fügen Sie eine Beschreibung hinzu, was der Besucher erwarten kann.
- Ein Preisschild für das Produkt (vielleicht sogar einen alten Preis anzeigen (Tipp: `<del>`)).
- Ein Bild des Produkts / (Bonus: Video einbetten, Gifs verwenden).
- Gestalten Sie den Text mit HTML-Tags, wo es angebracht ist (`<b>`, `<i>`, `<u>`, `<del>`, `mark`, `<sub>`/`<sup>`).
- Verlinken Sie auf eine andere von Ihnen erstellte Seite (und eine externe Seite) über das Link-Element (`<a>`).
- Fügen Sie außerdem einen Button hinzu, der über die ganze Seite geht und auf dem "Kaufen" steht.
- Erstellen Sie eine Liste der Inhalte, die man von dem Produkt erwarten kann (Bonus: Neben einer ungeordneten Liste können Sie auch eine geordnete Liste verwenden).

**Regel:** KEIN CSS ODER JAVASCRIPT ERLAUBT!! Versuchen Sie, so viel wie möglich aus reinem HTML herauszuholen.

### Für Neugierige

- Entdecken Sie andere HTML-Tags, um nur einige zu nennen: `<hr>`, `<details>`/`<summary>`, Image-`<map>`, `<iframe>`, `<meter>`, `<video>`.
- Recherchieren Sie, wie man Formularelemente erstellt, und erstellen Sie ein Formular, um sich für Ihren Spam-Newsletter anzumelden.
- Fragen zum Nachforschen:
    - Wie stellt man einen Text wie `<h1>` (als Text, nicht als HTML interpretiert!) in HTML dar?
    - Was hat Barrierefreiheit mit HTML zu tun, was sind semantische Tags?
