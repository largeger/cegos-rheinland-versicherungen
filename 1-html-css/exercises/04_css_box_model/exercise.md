# 📦 Übung: Das Box-Modell verstehen

## Aufgabe: Das Box-Modell untersuchen

Schauen Sie sich die Datei `cssbox.html` an. Diese Übung hilft Ihnen, die Abstände und Rahmen von Elementen zu verstehen.

### Schritte

1. Was ist der Zweck von `margin`, `padding` und `border`? Experimentieren Sie mit den Werten in der CSS-Datei.
2. Fügen Sie dem `body` ein zweites `div`-Element mit der Klasse `box` hinzu.
3. **Beobachtung:** Was passiert mit dem Außenabstand (`margin`) zwischen den Elementen?
4. **Layout-Wechsel:** Setzen Sie das CSS-Attribut `display` so, dass beide Boxen in derselben Zeile stehen (erinnern Sie sich an Inline- vs. Block-Elemente).
5. Wie verhält sich nun `margin` und `padding`?

## Aufgabe: Fortgeschrittenes Box-Modell

1. **Box-Sizing:** Das Attribut `box-sizing` ist standardmäßig auf `content-box` eingestellt. Was passiert, wenn Sie es auf `border-box` umstellen? Was genau ist der Unterschied in der Breitenberechnung?
2. **Bonus:** Wie können Sie den Rahmen nur auf einer Seite ändern (z. B. nur `border-left`)?
