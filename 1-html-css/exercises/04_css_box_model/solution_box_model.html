Erklärungen zur Lösung

  • margin: Erzeugt Außenabstand zwischen den Boxen. Bei display: inline-block addieren sich die seitlichen Margins (20px + 20px = 40px Abstand).
  • padding: Erzeugt Innenabstand zwischen dem Text "Box 1" und dem blauen Rahmen.
  • box-sizing: border-box: Sorgt dafür, dass die Box trotz Padding und Border exakt 200px breit bleibt. Ohne dieses Attribut (Standard content-box) wäre die Box 200 + 40 (Padding) + 20 (Border) = 260px breit.
  • display: inline-block: Ermöglicht es, dass Boxen nebeneinander stehen (wie inline), aber dennoch Breite, Höhe und vertikale Abstände akzeptieren (wie block).
