## Exercises: Conditional Rendering 

Some easy ideas to apply conditional rendering - feel free to choose one!

1. Create a component that displays "Good morning" if it's before 12 PM and "Good afternoon" otherwise. 🌞
---
2. Build a toggle button that changes its text between "On" and "Off" when clicked. 🔘
---
